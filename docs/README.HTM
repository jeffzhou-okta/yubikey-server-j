Description

YubiKey Validation & Management Server was first developed by Yubico and Yubico made it an open source project starting from June, 2008. Anyone that likes to participate the development work please contact send your bio, availability, modules you are interested to work on to Tech@Yubico.com. Yubico will continue coordinate the community efforts and make enhancements in new releases regularly.

Project Status

  • Nov. 2007 - Production launch, hosted by Yubico for the public to use for YubiKey OTP validation.
  • Jun. 2008 - Open source to the public

Current Priorities

  • Port to PostgreSQL to avoid licensing issue of using MySQL for commercial purposes
  • Test, migrate to use JDK 1.6 or latest
  • Associate YubiKeys to popular user repositories such as Active Directory, LDAP, etc.
  • Token management system features such as issuance tracking, reporting, temporary password, etc.
